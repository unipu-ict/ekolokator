# EkoLokator
Mobilna aplikacija za geolociranje, slikanje i evidentiranje ekološki problematičnih točaka u okolišu uz mogućnost prijave istih email-om nadležnim institucijama.

####Napomena
Trenutna verzija ne radi na Android 6.0!
